Conditions & usage

Coordonnées masquées

Sur le mur, les coordonnées (téléphone / email) restent masquées. Elles s’affichent uniquement après déblocage côté offreur.

Notes / avis

Après un échange réel, le demandeur peut laisser un avis pour aider la communauté.

Règles simples
  • Respect et langage correct (sinon suppression).
  • Pas de spam / pas de demandes illégales.
  • Chaque demande est destinée à recevoir plusieurs devis.
